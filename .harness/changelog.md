# 📝 변경 이력 (Changelog)

## [1.0.3] - 2026-06-17 - [주체: 💻 Frontend Developer]
- **작업 내용:** 토스트 안내 메시지 가독성 구분선(toast-divider) 추가
  - `src/App.jsx` 및 `src/App.css`를 수정하여 검색 결과 토스트 팝업 내 좌석 번호와 층 구분 정보를 분리하는 얇은 세로 구분선(1px vertical divider) 스타일 및 마크업 주입.
- **사유:** 좌석 식별자와 층 묘사 정보가 딱딱하게 연속해서 이어져 가독성을 해치던 사소한 UI/UX 결함을 개선하기 위함.
- **검증 결과:** UI 레이아웃 내 시각적 구분이 용이해짐을 육안 확인.

## [1.0.2] - 2026-06-17 - [주체: 👔 Project Manager]
- **작업 내용:** 프로젝트 활성 메모리(`.harness/memory.md`) 및 변경 이력(`.harness/changelog.md`) 구축 및 하네스 검증 실행
- **사유:** 에이전트 자가 검증 턴 훅 및 메모리 갱신 의무 지침을 준수하고, 작업 히스토리를 체계적으로 트래킹하기 위함.
- **검증 결과:** `memory_manager.js` 및 `verify.js` 자가 검증 도구를 구동하여 정합성(Exit Code 0) 확보 완료.

## [1.0.1] - 2026-06-17 - [주체: 💻 Frontend Developer / 👔 Project Manager]
- **작업 내용:** 라이트 테마(밝은 모드) 가독성 보정 및 테마 로컬스토리지 영속화 구현
  - `src/App.css` 및 `src/index.css`를 수정하여 밝은 모드에서의 모달 텍스트 및 접힌 구역 토글 버튼 가독성 문제(흰색 글자로 표시되는 현상 등)를 해결.
  - `src/App.jsx`에서 라이트 테마 활성화 상태가 새로고침 후에도 유지되도록 로컬스토리지 상태 보존 및 복원 로직 추가.
- **사유:** 테마 변경 시 웹 접근성 HSL 컬러 대비율을 만족하고 UX의 일관성을 높이기 위함.
- **검증 결과:** 테마 상태 영속성 정상 작동 검증 및 텍스트 색상 대비 개선 확인.

## [1.0.0] - 2026-06-16 - [주체: 👔 Project Manager / 🛡️ Gitignore Guard]
- **작업 내용:** `.gitignore` 파일 구성 정비 및 불필요한 대용량/임시 파일 추적 정리
  - `.gitignore` 설정을 보완하여 `node_modules`, `dist` 등 빌드 산출물과 IDE 캐시 차단.
  - 불필요하게 추적되고 있던 임시 엑셀 파일(배치도 copy.xlsx)을 저장소에서 안전하게 분리 및 제거.
- **사유:** 저장소 용량 낭비 방지 및 형상 관리의 무결성을 지키기 위함.
- **검증 결과:** `git status` 명령어로 정상 적용 상태 확인 및 원치 않는 추적 파일 제거 검증.
